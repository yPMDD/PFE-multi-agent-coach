Create a **flowchart** showing the complete LangGraph state machine for the PFE Coach multi-agent system.

## Requirements

Create a **Mermaid flowchart** showing the state graph structure:

### Graph Structure

**Start Node**
- START (entry point)

**Supervisor Node**
- Supervisor (main routing node)

**Conditional Router**
- Router (conditional edges based on intent)

**Agent Nodes (4 agents)**
- RAG Agent (intent: rag)
- Writing Coach (intent: writing_coach)
- Progress Tracker (intent: progress_tracker)
- General (intent: general)

**End Node**
- END

### Conditional Edges from Supervisor

- intent = "rag" → RAG Agent
- intent = "writing_coach" → Writing Coach

- intent = "general" → General (fallback)

```
START → Supervisor → [Conditional Router based on intent] →

```
